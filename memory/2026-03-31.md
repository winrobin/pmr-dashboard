# 2026-03-31 记忆日志

## PMR V5 投决级报告系统完成

### 核心升级
- **子评分体系**：5 维度（incomeStability/growthPotential/contractSecurity/dataCompleteness/operationalSimplicity）
- **总分公式**：`incomeStability*0.25 + growthPotential*0.25 + contractSecurity*0.20 + dataCompleteness*0.15 + operationalSimplicity*0.15`
- **verificationStatus**：verified / partial / listing-only / rumour / unknown
- **verificationDetails**：broker / bodyCorporate / financials 三字段布尔值
- **dataCompletenessScore**：0-100 可机器筛选
- **hardFailReasons**：直接淘汰条件数组（如 over-reliance-on-salary, letting-income-extremely-low）
- **nextAction 结构化**：actionType / requiredDocs / contactPerson / deadline / priority / blockingIssues
- **sourceEvidence**：逐字段证据链（sourceType/sourceDate/sourceUrl/quoteSnippet）
- **auditTrail**：变更历史追踪

### 统一命名规范
- `pmr-data-latest.json` — 永远指向最新数据（避免版本号断链）
- `pmr-report-latest.html` — 永远指向最新报告
- `pmr-schema-v5.json` — V5 数据模式定义

### HTML 交互功能
- 滑块筛选前 N 名（1-10 可调）
- 开关"仅显示已核实数据"
- 双表格展示：✅ 已确认 vs ⏳ 待核实
- 颜色编码：🟢 良好 / 🟡 中等 / 🔴 风险
- 验证徽章：✅ broker / ⚠️ partial / ❓ rumour
